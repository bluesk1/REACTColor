Installation (using REACTInstaller) Recommended

     1. Install Python from here

    2. Download the executable file here

    3. Give permission by executing: chmod +x path/to/downloaded/file

    4. Execute the file by double-tapping it, and you are done!

Installation (using Command Line)

     1. Install Python from here

    2. Run this code in the Terminal: pip install https://github.com/bluesk1/REACTColor/releases/download/v.0.0.1/REACTColor-0.0.1-py3-none-any.whl

    3. You are done!
